Turning Scene Photos Into Anime Art
→

This is not a map project, but I think it is too cool to exclude it from my project collection. Inspired from the popular movie Your Name, by Shikai Makoto, our group came up with idea that, we can transfer scene photos into this same anime style image, bringing more blue sky into the background. This project involves with object detection and image processing. I mainly used Matlab to alter the raster dataset. Our group applied multiple filters to mimic the anime effect. The outcome looks amazing!

 

Tools: Matlab for the image processing, HTML/CSS, JS for project website

Challenges: Filters, Image Segmentation, Object Detection
