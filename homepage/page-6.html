US Energy Consumption Per State 2016
→

This Interactive Map shows state profiles and energy estimates in 2016. It includes information of energy production, consumption, expenditures and ranking in each categories. When user selects from the dropdown menu, the corresponding bars, map and color scheme will change. The label and bar will be highlighted when users hover over each state. It’s fun to see the animation effect of D3 !

 

Tools: D3, HTML, CSS, Javascript

Challenges: Information Extraction, Chart Configuration
